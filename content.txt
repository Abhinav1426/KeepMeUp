A few thoughts on writing well. Good writing is clear before it is clever. Short
sentences are easier to follow than long ones. Prefer simple words over fancy
ones, and cut anything that does not earn its place. Read your draft out loud and
you will hear the parts that need fixing. Above all, respect the reader's time.


Notes on planning a trip

Start by deciding on the dates and the budget, since everything else depends on
those two things. Next, book the flights early, because prices tend to climb as
the date gets closer. Choose a place to stay that is close to the things you
actually want to do, and read recent reviews before you commit. Leave room in
the schedule for rest; a trip packed too tightly stops being a vacation.

def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    return merge(left, right)


async function fetchWithRetry(url, options = {}, retries = 3) {
  for (let attempt = 1; attempt <= retries; attempt++) {
    try {
      const res = await fetch(url, options);
      if (!res.ok) throw new Error(`HTTP ${res.status}`);
      return await res.json();
    } catch (err) {
      if (attempt === retries) throw err;
      await new Promise(r => setTimeout(r, 2 ** attempt * 100));
    }
  }
}


-- monthly revenue by customer
SELECT
    DATE_TRUNC('month', o.created_at) AS month,
    c.name,
    SUM(o.total_cents) / 100.0 AS revenue
FROM orders o
JOIN customers c ON c.id = o.customer_id
WHERE o.status = 'completed'
GROUP BY 1, 2
ORDER BY month DESC, revenue DESC
LIMIT 100;
